About This Project

Diabetes Prediction System using Machine Learning

Objective

Diabetes mellitus is one of the most common chronic diseases worldwide, affecting over 422 million people. Early detection is crucial to prevent serious complications such as heart disease, kidney failure, and vision loss.

This system leverages machine learning to predict the risk of diabetes based on medical parameters. By comparing multiple algorithms and using an ensemble approach, it provides accurate and transparent predictions to support early screening.

Dataset

The Pima Indians Diabetes Database originates from the National Institute of Diabetes and Digestive and Kidney Diseases. It contains diagnostic measurements from 768 female patients of Pima Indian heritage, aged 21 and above.

Models Used

Logistic Regression

A statistical model that uses a logistic function to model binary outcomes. It estimates the probability of diabetes by fitting a linear decision boundary to the input features.

Random Forest

An ensemble of decision trees that votes on the final prediction. Each tree is trained on a random subset of features and data, reducing overfitting and improving accuracy.

Support Vector Machine

Finds the optimal hyperplane that maximally separates diabetic and non-diabetic cases in high-dimensional feature space using a radial basis function kernel.

K-Nearest Neighbors

Classifies new cases based on the majority vote of the 11 nearest training samples, weighted by distance. Closer neighbors have more influence on the prediction.

XGBoost

An optimized gradient boosting algorithm that builds trees sequentially, where each new tree corrects errors from the previous ones. Known for high performance in structured data tasks.
